RunElevated

Programm mit erhöhten Rechten ausführen auf einem UAC System

Programme, welche keine erhöhte Privilegien anfordern, sie aber mit erweiterte Rechten starten wollen, können RunElevated benutzen.

RunElevated fordert diese erweiterten Administratorrechte für andere Programme oder Skripte an.
Erweitere das Privileg einer Batchdatei, der Kommandozeile cmd.exe, einem Dateimanager, Scripte, Befehle wie net.exe, ältere Programme oder jeder anderen Anwendung.

Sie können dieses Programm per Drag and Drop auf RunElevated.exe ziehen,
oder sie erstellen eine Verknüpfung oder eine Batchdatei mit dem Befehl
>> RunElevated.exe Application.exe <<

Beispiele:

  1. Starte die Kommandozeile cmd.exe mit erhöhten Rechten aus einem Administrator Account aus
    >> RunElevated.exe %windir%\system32\cmd.exe<<
  2. Starte eine Batchdatei oder ein anderes Script mit erhöhten Rechten aus einem Administrator Account
    >> RunElevated.exe c:\pfad\yourScript.bat <<
  3. Rufe den Befehl net.exe mit erhöhten Rechten auf um einen Benutzeraccount hinzuzufügen.
    >> RunElevated.exe NET USER username "password" /ADD <<
  4. Starte die Kommandozeile von einem nicht Admin-Account herauls als Administrator und mit erhöhten Rechten
    Dazu benötigen sie RunAsSpc.
    >> runasspc.exe /program:"c:\path\RunElevated.exe" /param:"%windir%\system32\cmd.exe" /user:"administrator" /password:"pass" <<
  5. Starte die Kommandozeile aus einem nicht Admin-Account heraus als Administrator und mit erhöhten Rechten über eine verschlüsselte Datei.
    Dazu benötigen Sie RunAsRob.exe.
    - Erstellen Sie eine verschlüsselte Datei in RunAsRob wie im Screenshot unten gezeigt.
    - Starten sie diese verschlüsselte Datei mit dem Befehl:
    >> RunAsRob.exe /cryptfile:"c:\temp\CommandLineElevated.xus" <<
