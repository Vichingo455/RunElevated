RunAsRob run as service

Run as Service startet Programme versteckt im Hintergrund.

Starte Anwendung als Dienst nach dem Booten, bevor ein Benutzer sich anmeldet, unter dem Systemkonto mit den höchsten Systemrechten.

Hilfreich für Hintergrundaufgaben wie Monitoring, Kopieraktionen oder Prüfroutinen auf einem Computer. Nützlich wenn die Anwendung mit höchsten Rechten starten soll und dabei keine Benutzeraktion erforderlich ist. Weil das Programm als Windows Dienst im unsichtbar im Hintergrund läuft, muss das entsprechende Programm seine Arbeit erledigen ohne irgendeine Benutzereingabe.

RunAsService von RunAsRob ist ein kleines Tool um ein Programm oder ein Script, wie eine Batchdatei, als Windows Dienst laufen zu lassen.
Die Methode ist einfach zu verstehen.
Nachdem der Dienst von RunAsRob über die Grafische Benutzerschnittstelle RunAsService.exe installiert wurde, können sie die Anwendung oder das Script hinzufügen, welches sie als Windows Dienst starten wollen.
Beim nächsten Booten des Computers, startet der Dienst von RunAsRob und liest in seinem Registry Pfad, welche Anwendungen von RunAsService gesetzt wurden und führt jedes einzelne Programm oder Script als Dienst unter dem Systemkonto aus.

Schnellanleitung:

  1. Starten Sie RunAsService.exe als lokaler Administrator.
  2. Drücken Sie den Button >> Install RunAsRob <<.
  3. Wählen Sie über >> Add application << die Anwendung aus, welche als Dienst laufen soll.
  4. Fertig.
  5. Nach einem Neustart des Computers (oder des Dienstes RunasRob) startet die Anwendung als Dienst mit Systemrechten.
